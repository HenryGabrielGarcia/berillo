# Protótipo do Site Berillo - Empresa de Engenharia

## Visão Geral do Protótipo

O protótipo criado para a empresa Berillo segue uma estética profissional e moderna, inspirada no site da Certo Ambiental, mas com uma paleta de cores verde mais sóbria e fechada conforme solicitado.

## Estrutura do Site

### 1. Cabeçalho (Header)
- **Logo:** "BERILLO" com ícone estilizado (letra B)
- **Menu de Navegação:** HOME | SERVICES | PROJECTS | ABOUT US | CONTACT
- **Cor de Fundo:** Verde escuro (#1A522D)
- **Texto:** Branco para contraste

### 2. Seção Hero
- **Título Principal:** "INNOVATION IN ENGINEERING"
- **Subtítulo:** "Building Tomorrow, Today"
- **Botão CTA:** "DISCOVER MORE"
- **Fundo:** Imagem de cidade/construção com sobreposição verde
- **Estilo:** Texto branco sobre fundo escuro com transparência

### 3. Seção de Serviços
- **Título:** "SERVICES"
- **Cards de Serviço:** 3 cards principais
  - **Mechanical:** Ícone de engrenagem
  - **Electrical:** Ícone de circuito
  - **Civil:** Ícone de estrutura/ponte
- **Layout:** Cards brancos com ícones verdes e texto descritivo
- **Fundo:** Cinza claro (#F5F5F5)

### 4. Seção de Contato
- **Título:** "CONTACT US"
- **Formulário:** Campos para Name, Email, Message
- **Informações de Contato:** Telefone, Email, Endereço
- **Mapa:** Pequeno mapa indicativo
- **Botão:** "SEND" em verde

### 5. Rodapé (Footer)
- **Nome da Empresa:** "BERILLO ENGINEERING"
- **Copyright:** "© 2024 All Rights Reserved"
- **Cor de Fundo:** Verde escuro (#1A522D)

## Paleta de Cores Aplicada

- **Verde Principal:** #2E8B57 (Sea Green) - Botões e destaques
- **Verde Escuro:** #1A522D - Header e footer
- **Cinza Claro:** #F5F5F5 - Fundos de seção
- **Cinza Escuro:** #333333 - Texto principal
- **Branco:** #FFFFFF - Cards e texto sobre fundos escuros

## Tecnologias Propostas

### HTML5
- Estrutura semântica com tags apropriadas
- Formulários com validação
- Meta tags para SEO e responsividade

### CSS3
- Flexbox/Grid para layout responsivo
- Transições e animações suaves
- Media queries para diferentes dispositivos
- Variáveis CSS para cores consistentes

### JavaScript
- Interatividade do menu
- Validação de formulário
- Animações de scroll
- Efeitos hover nos cards
- Menu mobile responsivo

## Funcionalidades Planejadas

1. **Menu Responsivo:** Hamburger menu para dispositivos móveis
2. **Smooth Scrolling:** Navegação suave entre seções
3. **Formulário de Contato:** Com validação em tempo real
4. **Animações:** Fade-in dos elementos ao fazer scroll
5. **Hover Effects:** Nos cards de serviço e botões
6. **Loading States:** Para envio de formulário

## Próximos Passos

1. Criar a estrutura HTML básica
2. Desenvolver o CSS com a paleta de cores definida
3. Implementar JavaScript para interatividade
4. Testar responsividade em diferentes dispositivos
5. Otimizar performance e acessibilidade

Este protótipo mantém a elegância e profissionalismo do site de referência, mas com uma identidade visual própria usando tons de verde mais sóbrios e fechados, perfeitos para uma empresa de engenharia.
